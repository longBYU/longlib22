This directory contains source code for the LONGLIB graphics subroutine library.
The subdirectories of this directory include:

c/  : c code
f/  : fortran code
README.txt : this file
tekoff, tekon : scripts to dump escape codes to the X windows XTERM
  program to put the program into the textronics graphics mode (tekon) or
  out of the graphics mode (tekoff) and back into the ordinary text mode.

The archaic nature of the code and its operation are acknowledged.

Copyright info:  Longlib was originally written by an employee of a US
government contractor and is in the public domain.
