This directory tree contains the LONGLIB3 graphics subroutine library.
The subdirectories of this directory include:

The main directory:

data/ : data files for map library and postscript converter
src/ : source code for the c (c/) and f (f/) versions of the library
doc/ : documentation
src/f/ : fortran code
src/c/ : c code
README.txt : this file

tekoff, tekon : scripts to dump escape codes to the X windows XTERM
program to put the program into the textronics graphics mode (tekon) or
out of the graphics mode (tekoff) and back into the ordinary text mode.
